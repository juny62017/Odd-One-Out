# Odd One Out

Odd One Out is a fun game where players must quickly find and click the box with a slightly different color before time runs out.
it  may look that all tiles are same but trust me.....
THERE IS 1 DIFFERENT COLOR !!!

## Features

- Random color generation
- Score tracking after guessing each tile correctly
- Level increases as you do well
- Countdown timer
- Best score system so that you can be proud of yourself 
- Increasing difficulty so you can be down to earth 
- Game over screen
- Restart game option
- Smooth animations

## How To Play

1. Click the Start Game button
2. Look carefully at the grid.
3. Find the box with a slightly different color.
4. Click the correct box to earn points.
5. Each correct answer increases your level.
6. As levels increase... the color difference becomes harder to spot.
7. Try to get the highest score before the timer reaches zero.
